# Enhance Prompt for Stitch

Transforms rough ideas into structured prompts for better Stitch results.

## Steps
1. **Assess**: Check for platform, page type, and visual style.
2. **DESIGN.md**: Incorporate tokens from the project's design system if available.
3. **Refine**: Replace vague terms with professional UI/UX terminology.
4. **Structure**: Organize into numbered sections (Header, Hero, Content, Footer).

## Example
"Login page" -> Styled, structured prompt with specific hex codes and layout description.
