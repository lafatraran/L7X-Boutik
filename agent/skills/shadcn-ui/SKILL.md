# shadcn/ui Integration

Discover, install, and customize shadcn/ui following ownership principles.

## Use
- **CLI**: `npx shadcn@latest add [component]`
- **Ownership**: Code is copied to your project, allowing full customization.
- **Theming**: Use CSS variables and Tailwind config for consistent styling.
