# Stitch to Remotion Walkthrough

Creates engaging videos from app designs with smooth transitions and zooms.

## Process
1. **Retrieve**: Get screen assets and metadata from Stitch project.
2. **Setup**: Configure Remotion project and manifest (`screens.json`).
3. **Compose**: Use `ScreenSlide` and transitions (Fade, Slide, Zoom).
4. **Render**: Output MP4/WebP video using Remotion CLI or MCP.
