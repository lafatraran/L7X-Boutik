# Stitch Build Loop

Autonomous iteratively site-building using a "baton" system (`next-prompt.md`).

## Iteration
1. Read the next prompt from the baton.
2. Generate page with Stitch.
3. Integrate into site structure and update docs.
4. Prepare the next baton to continue the loop.
